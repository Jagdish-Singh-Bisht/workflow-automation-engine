# Workflow Automation Engine

## Overview
A dynamic workflow automation engine buit using Spring Boot.
Workflows and nodes are stored in the database and executed dynamically.

## Features Implemented
- User entity
- Workflow entity
- WorkflowNode entity
- Execution logging
- Dynamic node-based execution engine
- REST API endpoints

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Lombok

## Future Enhancements
- Real AI API integration
- Authentication & Authorization (JWT)
- Node configuration support
- UI builder for workflows
